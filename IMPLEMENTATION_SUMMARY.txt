KEY TECHNICAL DETAILS
================================================================================

THE SECRET TO PROFESSIONAL MIXING - EQ Automation:

def crossfade_with_eq(a, b):
    # 1. Cut bass on outgoing track (prevents mud)
    a_filtered = eqffmpeg.low_pass(frequency=100.0, q=0.7, a)
    
    # 2. Remove rumble from incoming track (improves clarity)
    b_filtered = eqffmpeg.high_pass(frequency=50.0, q=0.7, b)
    
    # 3. Use sine-curve fades (4 seconds = natural sounding)
    fade_out = fade.out(type="sin", duration=4.0, a_filtered)
    fade_in = fade.in(type="sin", duration=4.0, b_filtered)
    
    # 4. No normalization (prevents clipping)
    return add(normalize=false, [fade_out, fade_in])

RESULT: Bass frequencies don't clash, incoming track is clear,
fading sounds natural. Professional DJ software quality.

   - Could enhance: Frequency-based compatibility (not just keys)
   - Future: Add frequency clash detection

3. Cue Detection: Hybrid energy+spectral method
